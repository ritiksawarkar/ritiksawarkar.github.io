# ⚙️ Running Locally

Clone the repository:

```
git clone https://github.com/ritiksawarkar/ritiksawarkar.github.io.git
```

Navigate to the project:

```
cd ritiksawarkar.github.io
```

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

Build production version:

```
npm run build
```
